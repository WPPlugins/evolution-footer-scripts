== Description ==

This small plugin moves all scripts (including jQuery) to the footer to help speed up page load times, while keep stylesheets in the header. Note that this only works if you have plugins and a theme that utilizes wp_enqueue_scripts correctly.

== Installation ==

1. Upload `evolution-footer-scripts.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Once activated, it should work.
